## #开发环境搭建

**URL:** https://book.makepad.rs/zh/contribute/dev-setup

**Contents:**
- #开发环境搭建
- #准备工作
  - #安装 Rust

Makepad之书 使用 Rust 和 Rspress 来构建。

Makepad之书 使用 pnpm 作为包管理工具。

若你已经安装了 npm、yarn，你可以使用 npm 或 yarn 来安装pnpm。

**Examples:**

Example 1 (unknown):
```unknown
1npm install -g pnpm
```

Example 2 (bash):
```bash
1pnpm install  # 安装依赖
2pnpm run dev  # 启动项目
```
